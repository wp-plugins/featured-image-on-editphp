Displays featured images on edit.php.

== Description ==
When you visit edit.php, you will see the list of posts. WordPress 3.0 has a nice feature called featured image, but not shown on edit.php. This plugin diplays featured images in the list of posts.

== Installation ==
Copy the featured_image_on_edit.php into your plugin directoy, and activate it.

== Frequently Asked Questions ==
= I want to change the size of images =
In version 1.0, the size of images is fixed to thumbnail(150x150). If you want to change it, find tha  the_post_thumbnail('thumbnail'); in featured_image_on_edit.php and modify it.
